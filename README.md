Vim gitignore plugin
====================

* This plugin provides syntax highlighting for `.gitignore` file.


Requirements
------------

None


Installation
------------

* Using [vundle][4], or [neobundle][5], or other options.

* [vundle][4]

    ```VimL
     Plugin 'gisphm/vim-gitignore'
    ```

* [neobundle][5]

    ```VimL
     NeoBundle 'gisphm/vim-gitignore'
    ```
